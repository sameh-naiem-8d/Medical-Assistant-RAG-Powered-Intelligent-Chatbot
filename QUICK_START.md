# Quick Start

1. Install Python dependencies from `requirements.txt`.
2. Run `python scripts/setup_from_local_archive.py --archive "D:/Project Graduation/MEDBRIDGE_FINAL_AI_DELIVERY"`.
3. Run `python scripts/verify_artifacts.py`.
4. Start backend with `scripts/run_backend.ps1 -Port 8011`.
5. Run `python scripts/smoke_test.py --base-url http://127.0.0.1:8011`.
