# Literature Review

## What is RAG?
Retrieval-Augmented Generation (RAG) is a technique that combines information retrieval with text generation.  
The system first retrieves relevant documents from a knowledge base, then uses a language model to generate accurate answers based on that data.

---

## Key Technologies
- **LangChain:** Framework for building LLM-powered applications  
- **FAISS:** Library for vector similarity search  
- **Azure Cognitive Search:** Cloud-based search service by Microsoft  
- **Hugging Face:** Platform for machine learning models
