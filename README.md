# Medical RAG Chatbot

## Description
An AI-powered chatbot that answers medical questions using Retrieval-Augmented Generation (RAG).

---

## Features
- Retrieval-based answers  
- Accurate responses using knowledge base  
- Fast and scalable system  

---

## Tech Stack
- Python  
- LangChain  
- FAISS  
- Hugging Face  
- Azure  

---

## How to Use
1. Clone the repository  
2. Install dependencies  
3. Run the backend server  
4. Send queries via API
